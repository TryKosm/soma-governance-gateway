# Security Policy

## Supported Versions

This repository is currently maintained on the `main` branch.

## Reporting a Vulnerability

Please do **not** open public issues for security reports.

Send details to: **security@trykosm.com**

Include:
- A clear description of the issue
- Reproduction steps or proof-of-concept
- Potential impact and affected endpoints
- Any suggested mitigation

## Response Targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Mitigation timeline: shared after triage

We will coordinate disclosure and credit responsible reporters where appropriate.
